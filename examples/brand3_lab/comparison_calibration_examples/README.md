# Comparison Calibration Examples

These examples show how local draft exports from the perceptual narrative comparison viewer can be normalized into calibration logs.

They are experimental lab artifacts only:

- no production persistence
- no scoring integration
- no prompt changes
- no report renderer changes
- no Visual Signature integration
- no official review records

Files:

- `viewer_export_draft_example.json`: raw local draft shape exported by the viewer.
- `normalized_single_reviewer_example.json`: one reviewer normalized into calibration format.
- `multi_reviewer_agreement_example.json`: three-reviewer case with agreement and unsafe veto behavior.
- `aggregate_calibration_summary_example.json`: summary-level trends for a calibration run.
