# Brand3 Perceptual Library

This directory holds static, curated perceptual examples for Brand3.

The library is not connected to runtime behavior. Files here do not change scoring, prompts, reports, Visual Signature, or ingestion.

## Current Corpus

The initial corpus is the public FLOC* Work section:

- Source: https://wearefloc.com/work
- Index: `floc_work_corpus_index.md`
- Machine-readable index: `floc_work_corpus_index.json`
- First case artifact: `cases/charms/`

## Use

Use this library to study visible brand surface behavior:

- surface behavior
- pacing
- visual rhythm
- editorial tone
- hierarchy
- interaction feeling
- density versus silence
- perceived sophistication
- perceived intentionality
- ritualized interface behavior
- emotional temperature
- product and UX language
- motion and interaction language

## Constraints

- Public website only.
- No private Notion.
- No runtime integration.
- No scoring.
- No prompt changes.
- No Visual Signature changes.
- No ingestion pipeline.
- No deep scraping unless a later task explicitly requests it.

## Current Status

This is a shallow corpus index. It records public case links, public descriptions, visible service tags, expected perceptual relevance, and suitability flags for later manual extraction.
